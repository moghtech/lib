# moghtech/lib

Collection of Rust and Typescript libraries used across Mogh apps.
These libraries handle common web app utilities, ensuring consistency across the Mogh ecosystem.

### APIs

- mogh_auth_client + mogh_auth_server

### Rust crates

- mogh_cache
- mogh_config
- mogh_error
- mogh_logger
- mogh_rate_limit
- mogh_request_ip
- mogh_secret_file
- mogh_server
- mogh_validations

### Typescript packages

- @moghtech/monaco-editor
- @moghtech/xterm
