# moghtech/lib

Collection of Rust and React (Typescript) libraries used across Mogh apps.
These libraries handle common web app utilities, ensuring consistency across the Mogh ecosystem.

### Rust crates

- mogh_cache
- mogh_config
- mogh_logger
- mogh_rate_limit
- mogh_secret_file
- mogh_error

### Typescript packages

- @moghtech/monaco-editor
- @moghtech/xterm
